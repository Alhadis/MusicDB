A collection of scripts for maintaining my music library, linting it for
oddities, and wrangling iTunes[^1] libraries exported as XML.

[^1]: I adamantly refuse to call this program “Apple Music”. I've called
	it iTunes since 2008, and I'll damn well keep doing so for as long
	as I'm forced to use macOS.
